# Flappy Bird Clone

## Overview

A fully functional Flappy Bird clone built with React and Canvas 2D rendering.

## User Preferences

Preferred communication style: Simple, everyday language.

## Project Architecture

- **Frontend**: React 18 + TypeScript + Vite + Canvas 2D
- **Backend**: Express.js + TypeScript
- **Game Component**: `client/src/components/FlappyBird.tsx`
- **Entry Point**: `client/src/App.tsx`

## Key Features

- Classic Flappy Bird physics and gameplay
- Web Audio API sound effects
- localStorage for best score and mute preference
- High-DPI display support
- Touch and keyboard controls
